Tipe Perilaku Objek
Solid — Platform yang bisa diinjak (tangga, jembatan, papan)
Danger — Objek berbahaya, menyebabkan kegagalan (pedang, pisau, paku)

Prinsip Arsitektur
AI inference 100% di browser — tidak ada server dalam loop inferensi
Momen keputusan HITL (Fase 3) adalah inti inovasi sistem
Server HANYA menerima log, TIDAK PERNAH mengirim data kembali ke client
Tiga kategori perilaku (Solid/Danger/Decoration) → tiga hasil gameplay
