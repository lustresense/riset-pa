# Agent Protocol

This repository uses a lightweight Recursive Self-Improvement workflow.

Before making changes, every AI agent must:

1. Read `MEMORY.md`.
2. Respect the 4-zone repository structure.
3. Keep root clean: only project-level documentation and config files belong there.
4. Use `00_RESEARCH/ideation/` for ambiguous files.
5. Update `MEMORY.md` and `CHANGELOG.md` after major structural or research changes.

Do not move or delete `MEMORY.md`, `CHANGELOG.md`, `.agent_protocol.md`, or
`README.md` without replacing them with an updated root-level version.
