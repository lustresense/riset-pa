2 FASE SIKLUS BERMAIN — Tiga Level Progresif
L1: Percaya Diri
→
L2: Bingung

L3: Salah
L1 "Turun dari Tebing" — Pembangunan Kepercayaan
Adegan: Stickman di tepi tebing, harus turun
Momo meminta:
"Gambar TANGGA biar Stickman bisa turun!"
Pemain menggambar → AI mengklasifikasi
Momo menjawab: Percaya Diri
"Aku 96% yakin ini tangga!"
Gelembung hijau — kepercayaan tinggi, klasifikasi benar
KEPUTUSAN: TERIMA → Berhasil
Tangga muncul → Stickman berjalan turun → BERHASIL
Tidak ada jalur gagal di Level 1 — membangun kepercayaan dasar pemain terhadap AI
L2 "Pintu Kunci" — Menavigasi Ambiguitas
Adegan: Dua pintu, satu aman satu terjebak
Momo meminta:
"Gambar KUNCI untuk buka pintu!"
Pemain menggambar → AI bingung mengklasifikasi
Momo ragu: Bingung
"Hmm... 62% kunci, 58% palu"
Gelembung kuning — kepercayaan terbelah, dua klasifikasi berdekatan
Keputusan Pemain:
KOREKSI → Kunci (Benar)
Pintu aman terbuka → Stickman lewat dengan selamat
✦ BERHASIL
TERIMA → Palu (Salah)
Pintu jebakan terbuka → Stickman terpukul
✕ KERTAS SOBEK!
Momo panik:
"Aduh! Kertas sobek! Coba koreksi tebakan gue!"
Coba Ulangi
Kembali ke fase menggambar → Override atau Koreksi → Berhasil
L3 "Hiasan Mematikan" — Koreksi Kritis
Adegan: Jalan terhalang patung hiasan indah (dekorasi)
Momo meminta (petunjuk terbuka):
"Gambar sesuatu untuk LEWAT"
Tidak ada petunjuk objek spesifik — pemain bebas menggambar apa saja
Pemain menggambar palu → AI salah klasifikasi
Momo salah tapi percaya diri: Salah
"Aku 51% yakin ini BUNGA!"
Gelembung merah — AI percaya diri tapi klasifikasi salah total
Keputusan Pemain:
OVERRIDE → Ketik "palu"
Palu muncul → Patung hancur → Jalan terbuka
✦ BERHASIL
TERIMA → Bunga (Salah)
Bunga hiasan muncul → Tidak ada efek
Stickman terjebak selamanya
✕ KERTAS SOBEK!
Momo menyesal:
"Gue... gue salah. Maaf. Tolong koreksi gue."
Coba Ulangi
Kembali ke fase menggambar → Override AI → Berhasil
