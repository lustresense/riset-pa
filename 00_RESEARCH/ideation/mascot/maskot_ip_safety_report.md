## 1. Methodology

### 1.1 IP Safety Testing Framework

Each design option is evaluated against 5 IP safety tests:

| Test | Description | Pass Criteria |
|------|-------------|---------------|
| **Silhouette Test** | Can the character be identified from a black silhouette alone? | Must NOT be confused with any existing character's silhouette |
| **Color Association Test** | Is the primary color associated with an existing famous character? | Must NOT use the iconic color of any well-known character as primary |
| **Concept Originality Test** | Is the underlying concept (origin, species, behavior) original? | Must NOT replicate the origin story or core concept of existing character |
| **Name Collision Test** | Does the name conflict with existing IP? | Must NOT be identical or confusingly similar to existing character names |
| **Derivative Test** | Could a reasonable person conclude this is a copy or modification of existing character? | Must be demonstrably original in at least 3 major aspects |
